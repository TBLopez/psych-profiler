# Psychological Profiler

A password-protected, AI-powered psychological profiling interview tool. Share a link with someone, they enter a password, go through a structured 5-phase interview, and receive a detailed 10+ page psychological profile report.

## Architecture

```
User's Browser ──HTTPS──▶ Cloudflare Worker ──▶ DeepSeek API
  (GitHub Pages)           (password check,       (your key,
                            AI proxy,               never exposed)
                            report generation)
```

- **Frontend:** Single HTML file hosted on GitHub Pages
- **Backend:** Cloudflare Worker (free tier) — validates password, proxies AI calls, generates reports
- **Your DeepSeek API key** — stored as Worker secret, never exposed to browser
- **Report:** Generated by AI, returned directly to the recipient's browser for download

## Quick Deploy

```bash
# 1. Deploy the Cloudflare Worker
cd backend
npm install
npx wrangler login
npx wrangler secret put PASSWORD    # Set: ForFarook2026!!
npx wrangler secret put DEEPSEEK_KEY # Set your DeepSeek API key
npx wrangler deploy

# 2. Update frontend with your worker URL
# Edit frontend/index.html — change API_BASE to your worker URL

# 3. Deploy frontend to GitHub Pages
cd ../frontend
git init
git add index.html
git commit -m "Add psych profiler"
# Push to GitHub, enable Pages
```

## Privacy

- API key is stored as a Cloudflare Worker secret — never in client-side code
- Conversation data flows directly between browser and DeepSeek via the worker
- Report is downloaded directly to the recipient's machine
- No data is stored on any server
- No analytics, no tracking, no third-party scripts
